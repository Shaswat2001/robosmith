# RoboSmith Docs

This is the Astro Starlight documentation site for RoboSmith.

```bash
npm install
npm run dev
npm run build
```

The production site is configured for GitHub Pages at:

```text
https://Shaswat2001.github.io/robosmith/
```

The source content lives in `src/content/docs/`, and the site navigation is in
`astro.config.mjs`.
